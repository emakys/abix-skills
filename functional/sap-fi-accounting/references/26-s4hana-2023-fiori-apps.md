## Nota tecnica: F0718 vs FB01

```
IMPORTANTE para validaciones y sustituciones:
  F0718 (Post General Journal Entries) NO ejecuta la transaccion FB01 internamente.
  Usa el Journal Entry API (BAPI_ACC_DOCUMENT_POST / ACC_DOCUMENT).

  Implicacion: validaciones configuradas con callup point que verifican SY-TCODE = 'FB01'
  NO se disparan al usar F0718. Hay que adaptar las validaciones para que funcionen
  independientemente del TCODE, o usar BAdIs que se ejecutan en ambos escenarios.
```

## Activacion Fiori FI en S/4HANA 2023

```
1. Roles de usuario:
   SU01 → asignar Business Role (SAP_BR_GL_ACCOUNTANT, SAP_BR_AP_ACCOUNTANT, etc.)

2. Catalogos:
   SPRO → Fiori → Reference Apps → Activate by Business Catalog

3. Rapid Activation:
   /UI2/FLPD_CONF → task list SAP_BASIS_FND_FIORI_2023

4. OData Services:
   /IWFND/MAINT_SERVICE → activar servicios por app ID

5. Verificar:
   /UI2/FLP → Fiori Launchpad → verificar tiles visibles por rol
```
